1. 주제 선정 동기와 이루고 싶은 목표

저는 소프트웨어학과에 재학 중이며, 데이터를 기반으로 현실 문제를 해결하는 기술에 큰 흥미를 가지고 있습니다. 편입 전 다니던 대학교의 캡스톤디자인 수업에서 스마트팜 프로젝트를 진행하면서, 실제 데이터를 수집하고 전송 및 시각화하는 전 과정을 직접 경험하게 되었고, 이 과정에서 빅데이터 기술에 큰 관심을 갖게 되었습니다. 그 경험 이후로 데이터를 통해 의미 있는 인사이트를 도출하는 일에 매력을 느껴, 현재는 데이터 엔지니어나 분석가를 목표로 공부하고 있습니다.

2. 본인 역량 성장 과정

이 프로젝트에서 저는 아두이노와 온습도 센서를 활용하여 식물 생장 데이터를 수집하고, 이를 Node.js 기반 서버에 소켓 통신을 통해 전송하는 작업을 맡았습니다. 특히, 교수님이 제공해주신 데모 장비를 기반으로 Unity와 연동하여 VR 환경에서 데이터를 실시간으로 시각화하는 기능을 구현해냈으며, 이 결과로 프로젝트 발표회에서 우수상을 수상할 수 있었습니다.

3. 중요 사회 이슈와 나의 견해

2025년 5월 발생한 SK텔레콤 유심(USIM) 해킹 사고는 수십만 명의 가입자 정보가 외부로 유출되었고, 인증 절차를 악용한 범죄로까지 이어진 중대한 보안 사고였습니다. 이 사건을 통해 기술이 아무리 고도화되더라도, 개인정보 보호와 데이터 윤리의 중요성이 간과되어선 안 된다는 점을 절실히 느꼈습니다. 특히 개발자의 역할이 단순히 기능 구현에 그치지 않고, 사용자 데이터를 안전하게 보호할 수 있는 구조를 설계하고 윤리적 기준을 갖추는 것까지 포함되어야 함을 깨달았습니다. 이에 저는 개발자로서 보안과 프라이버시를 핵심 가치로 삼고, 책임 있는 기술 개발을 실천하는 자세를 갖추고자 합니다.

4. 기술적으로 어려웠던 프로젝트와 해결 방법

캡스톤 경진대회를 준비하면서 여러 예기치 못한 기술적 문제에 직면했습니다. 발표 전날에는 정상적으로 작동하던 소켓 통신이 갑자기 중단되고, 습도 센서가 고장나는 등 위기 상황이 발생했습니다. 특히, 데모 스마트팜 모델로 수집한 식물 생장 데이터를 시각화하는 단계까지는 무사히 진행되었지만, 해당 데이터를 데이터베이스에 저장하는 과정에서 값이 누락되거나 비정상적으로 입력되는 문제가 발생해 3일 동안 해결하지 못한 적도 있었습니다. 저는 이때 침착하게 콘솔 로그를 활용해 데이터 흐름을 하나하나 추적하고, 쿼리를 반복 점검하면서 문제 원인을 파악했습니다. 최종적으로 값이 저장되는 중간 단계에서 데이터 형 변환이 발생한다는 사실을 발견했고, 이를 방지하기 위해 데이터 타입을 명시적으로 지정하는 방식으로 해결했습니다. 이러한 경험을 통해 저는 기술적인 문제를 끝까지 해결하려는 끈기와 문제 분석 능력, 위기 상황에서도 침착하게 대응하는 자세를 갖추게 되었습니다.

5. 본인 역량

저는 문제 해결 능력과 새로운 기술을 빠르게 습득하는 데 강점을 가지고 있습니다. 다양한 프로젝트 경험을 통해 HTML, CSS, JavaScript뿐만 아니라 Node.js, 소켓 통신, Unity 연동, 센서 제어 등 여러 기술을 실무적으로 활용해보았습니다. 특히 데이터를 수집하고 가공하여 의미 있는 결과로 시각화하는 전 과정을 직접 수행하면서, 데이터 파이프라인에 대한 이해와 실습 능력을 키웠습니다. 또한 프로젝트 발표와 팀워크를 통해 소통 능력과 책임감을 동시에 기를 수 있었으며, 기술뿐만 아니라 윤리적인 개발자의 자세 또한 중요하게 여기고 있습니다. 앞으로도 꾸준히 기술 역량을 향상시키고, 실질적인 가치를 만들어낼 수 있는 개발자가 되기 위해 노력하고 있습니다.

관심 기술 & 관심 기업

빅데이터

제가 빅데이터 기술에 관심을 갖게 된 계기는, 스마트팜 프로젝트에서 실제 데이터를 수집하고 시각화하는 과정을 경험하면서부터입니다. 센서를 통해 수집한 데이터를 서버로 전송하고, 그것을 실시간으로 시각화하는 작업을 직접 해보면서 데이터가 단순한 숫자가 아니라 문제 해결의 핵심이라는 점을 깨달았습니다. 빅데이터는 보통 데이터의 수집, 저장, 처리, 분석, 시각화 단계를 포함하며, 단순한 기능 구현을 넘어 의미 있는 결정을 이끌어내는 기술이라고 생각합니다. 저는 Python의 Pandas와 NumPy를 중심으로 기초 분석 역량을 키우고 있으며, 앞으로 Apache Spark, SQL, Tableau와 같은 도구들을 익혀나갈 계획입니다. 이러한 기술을 바탕으로 실제 산업 현장에서 데이터 기반의 문제 해결을 이끌 수 있는 사람이 되고 싶습니다.

판교 R&D 글로벌센터 & 메타파머스

판교 R&D 글로벌센터는 NHN, 더존비즈온, 네이버 등 다양한 IT 기업이 밀집한 국내 기술 중심지입니다. 이들 기업은 모두 빅데이터, 클라우드, AI 기술을 실무에 적극적으로 활용하고 있으며, 제가 경험한 스마트팜 프로젝트에서의 데이터 수집·분석 경험과도 기술적으로 연결된다고 생각합니다. 최근에는 스마트 농업과 AI 농업 기술에 특화된 기업인 메타파머스(Metafarmers)에도 관심을 갖고 있습니다. 메타파머스는 농업 데이터를 수집하고 이를 분석하여 작물 생장 환경을 자동 제어하거나 예측하는 솔루션을 개발하는 기업으로, 제가 프로젝트에서 다뤘던 센서 기반 환경 데이터 수집과 자동 제어 흐름과 매우 밀접한 연관이 있습니다. 향후에는 NHN Cloud, 더존비즈온, 메타파머스와 같은 기업에서 인턴십이나 공동 프로젝트에 참여하며, 데이터 기반 시스템을 실제 산업 환경에서 어떻게 안정적으로 구현하고 운영하는지를 배우고 실무에 기여하고 싶습니다.

Future Plan

앞으로 저는 Python 기반의 데이터 분석 능력을 심화하고, Spark와 SQL을 활용한 대용량 데이터 처리 역량을 확보할 예정입니다. 또한 Tableau, PowerBI 같은 시각화 도구를 활용해 데이터를 더 명확하고 직관적으로 전달할 수 있는 능력도 함께 키워나가고자 합니다. Kaggle이나 Dacon과 같은 공모전에 참가하여 실전 경험을 쌓고, NHN, 더존비즈온과 같은 기업에서 인턴십을 통해 실무 역량을 갖추는 것이 1차 목표입니다. 필요 시 대학원 진학도 고려하면서, 분석과 실무를 모두 아우르는 데이터 전문가로 성장해가고자 합니다.
